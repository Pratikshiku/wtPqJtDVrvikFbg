## SpringBoot and Vue Pet Adoption System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/250b57bca73148268a635f3cf5252648/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Pet Adoption
5. Stray Pet Rescue
6. Pet Feeding Points
7. Lost Pet Viewing
8. Stray Pet Rescue Station
9. Pet Forum
10. Pet Donation
11. Educational Articles
12. Popular Activities


**Administrator Functions:**
1. Login
2. Dashboard
3. User Management
4. Role Management
5. Menu Management
6. File Management
7. Stray Animal Management
8. Animals Pending Sterilization Management
9. Adoption Application Management
10. Comment Management
11. Stray Animal Rescue
12. Feeding Point Management
13. Lost Animal Management
14. Rescue Station Management
15. Post Management
16. Donation Management
17. Announcement Management
18. Educational Article Management
19. Activity Management
